= BlueXP 分類過時功能
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
BlueXP 分類是 BlueXP 中的核心功能、無需額外付費。NetApp 將 BlueXP 分類為所有客戶都能使用的核心 BlueXP 功能、讓您能夠使用核心功能來存取量身打造的資料管理功能。

BlueXP 核心版本從 1.31 版及更新版本開始、有些功能已經過時、但舊版 1.30 及更早版本仍支援這些功能。
